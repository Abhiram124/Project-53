React Native

What is React Native?

React Native is an open-source mobile application framework created by Facebook.It is used to develop applications Android,IOS,Web and UWP. Facebook,instagram,tesla,uber eats are some apps made in this framework.

About React Native

React Native is based on React.js, which is written in JavaScript.It is simple to learn. You have functions like 'import' and 'export'.Import function brings into app.js which is where the main code is located. You can create other files and simply import them to app.js.To add sound you have 'playSound' and for images it is 'image'.
